# homebrew-tap

Homebrew formulae for [aytzey](https://github.com/aytzey) tools.

## Install

```sh
brew install aytzey/tap/showagent
```

## Formulae

| Formula | Description |
| --- | --- |
| [showagent](https://github.com/aytzey/showagent) | Browse, resume, branch, and convert local AI coding sessions in one TUI |
